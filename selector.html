CSS SELECTORS

=>This is denoting the selector tag.

=>There are four types of selectors.

  1. simple selector
    • name

    • cls

      • starts with .

    • id

      • starts with #

  2. combinator selector
    • descandant selector

      • selects all the elements that are descendants to the specific element.

    • child selector

      • selects all the elements which are children of specific element.

    • general sibiling selector

      • select an element that is directly after another specific element or selects the elements that are placed immediately after the child elements.

    • adjacent sibiling selector

      • selects all elements that are next siblings of a specified element.

  3. pseudo-class selector
    • first-child Pseudo-class

      • higlights the first child of the parent ie., div-parent p- firstchild

    • first-child Pseudo-class with a tag in

      • highlights all the i tag of parent p tag.

    • first-child Pseudo-class with a tag out

      • higlights only the first child of p having i tag.

  4. pseudo-element selector
    • first-line Pseudo-element
    • first-letter Pseudo-element
    • after Pseudo-element
    • selection Pseudo-element
    • marker pseudo-element
  5. Attribute selectors
    • [attribute="value"]

      • specific attribute highlighted or called.

    • [attribute~="value"]

      • attribute's specific value is called or highlighted.

    • [attribute|="value"]

      • attribute having the exact specific value or continued with a hypen will be highlighted or called.

    • [attribute^="value"]

      • attribute which starts with specific value will be called or highlighted.

    • [attribute$="value"]

      • attribute which ends with the specific value will be called or highlighted.

    • [attribute*="value"]

      • attribute which contains the half of the specific valuewill be called or highlighted.
